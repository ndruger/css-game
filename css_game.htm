SCORE:
JavaScriptを使わず、CSSとHTMLで作ったクリックシューティングゲームです。
Chrome / Safariの最新版で動きます。IE / Firefoxでは動きません。

● 操作方法
- 移動する牛をクリックすると捕まえることができ、スコアが増えます。全部で101匹います。
- F5で再スタートできます。
